Handball ProStat V15 Elite
Handball ProStat V15 Elite je profesionalna aplikacija za rukometni skauting i statistiku, optimizovana za rad na tabletima.
📊 Funkcionalnosti
• Praćenje u realnom vremenu: Rezultat i sat utakmice.
• Interaktivna mapa šuta: Bilježenje tačne pozicije šuta na terenu.
• Analiza golmana: Mapiranje odbrana i primljenih golova.
• Efikasnost odbrane: Praćenje uspješnosti različitih formacija (6-0, 5-1, itd.).
• Zapisnik akcija: Identifikacija uigranih akcija koje rezultiraju golom.
• Eksport u PDF: Automatsko generisanje profesionalnog izvještaja.
🚀 Instalacija
Aplikacija se sastoji od jednog fajla, što je čini idealnom za offline rad.
1. Preuzmite index.html.
2. Otvorite fajl u pretraživaču (preporučujemo Chrome ili Safari).
3. Na tabletu koristite opciju "Add to Home Screen" za iskustvo prave aplikacije.
🛠 Tehnologije
• HTML5 / CSS3
• Vanilla JavaScript (Bez zavisnosti)
• SVG (Za taktiku i mapiranje)
• Tailwind CSS (Dizajn)
Developed for professional handball analytics.
